# Dentix360

Sistema de gestión de citas odontológicas desarrollado en Java.

## Descripción

Dentix360 es una aplicación que permite administrar citas en una clínica odontológica. 
El sistema permite registrar pacientes, registrar doctores, solicitar citas y cancelar citas.

## Tecnologías utilizadas

- Java
- MySQL
- JDBC
- NetBeans
- Maven
- Java Swing

## Funcionalidades

- Registro de pacientes
- Registro de doctores
- Solicitud de citas
- Cancelación de citas
- Consulta de citas
